# 152. 乘积最大子数组



代码如下

```java
class Solution {
    public int maxProduct(int[] nums) {
        int maxProduct = Integer.MIN_VALUE;
        int l = nums.length;
        for (int start = 0; start < l; start++) {
            int product = 1;
            for (int end = start; end < l; end++) {
                product *= nums[end];
                if (product > maxProduct) {
                    maxProduct = product;
                }
            }
        }
        return maxProduct;

    }
}
```
